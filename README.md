# Orbital Atlas 🛰️

Interactive 3D viewer of Earth's orbital environment — live-propagated satellites,
stations, and debris on a dark night-Earth globe. Rotate 360°, zoom, click any
object for details, filter layers, and trace orbit paths.

- **Data:** real catalog (Celestrak TLE), ~10–15k objects, propagated live (SGP4)
- **Stack:** Vite + TypeScript, Three.js, satellite.js, Web Worker
- **Status:** 🚧 In design → implementation

See the design spec: [`docs/superpowers/specs/2026-06-07-orbital-atlas-design.md`](docs/superpowers/specs/2026-06-07-orbital-atlas-design.md)
